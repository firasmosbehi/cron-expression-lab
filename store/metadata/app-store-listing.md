# App Store Listing Checklist

- Bundle ID: set in Xcode (`PRODUCT_BUNDLE_IDENTIFIER`, e.g., `com.yourcompany.cronexpression`).
- App Name: Cron Expression Generator & Tester
- Subtitle: "Generate & verify cron quickly"
- Keywords: cron, scheduler, dev tools, crontab
- Privacy policy URL: link to `store/privacy-policy.md` host.
- Privacy Nutrition Label: currently “no data collected”.
- Age rating: complete new questionnaire (Jan 31, 2026 update).
- App icon: 1024×1024, no alpha.
- Screenshots: 6.5" and 6.9" iPhone; include builder, verifier, results, dark mode.
- ATT: If you add ads/analytics that track, present ATT prompt; otherwise mark “No”.
- In-app purchase: if selling Pro, add a non-consumable product; reflect in UI.
- Export compliance: self-classify (typically “No encryption” → exempt).
