# Google Play Listing Checklist

- App name: Cron Expression Generator & Tester
- Package name: set `applicationId` in `android/app/build.gradle.kts` (e.g., `com.yourcompany.cronexpression`).
- Short description: "Pick a schedule, get the cron string, verify with upcoming run times."
- Full description: highlight presets, offline verification, dark mode, saved recipes (Pro), ads notice.
- Screenshots: at least 3 phone (1080×1920+); include builder, verifier, next-runs.
- Icon: 512×512 PNG, 32-bit with alpha.
- Feature graphic: 1024×500 PNG.
- Content rating questionnaire: required.
- Data safety form: reference `store/privacy-policy.md` (current: no data collected).
- Ads declaration: toggle on if you add AdMob/ads SDK.
- Target API: 35 (configured).
- App signing: prefer Play App Signing; otherwise upload your own keystore.
- Testing: set up Open/Closed testing first; review Pre-Launch Report.
